Python pass операторы

Python программалауында pass операторы - бұл келесі код үшін толтырылма көлемінде пайдалануға болатын бос оператор.

Мысалы, бізде өзгеше кодты реализациялау үшін жасалатын цикл немесе функция болса, бірақ біз оны келесіде реализациялау көмек көрміз көмекші көлемі ретінде pass операторын пайдалана аламыз.

pass операторынің синтаксисі:

pass

pass операторын шартты оператормен пайдалану

n = 10

# шартты оператор ішінде pass пайдалану
if n > 10:
    pass

print('Hello')

Мысалда, if операторы ішінде pass операторын пайдаланғанымызды көріңіз.

Бірақ pass іске асырылғанда ештеңе орындалмайды. Бұл нәтиже бермейді (NOP).

Егер біз pass пайдаланбаймыз немесе тек комментарий жазсақ болса:

n = 10

if n > 10:
    # кейінгіде код жазу
    pass

print('Hello')

Мысалда, бізге IndentationError: expected an indented block деген қате хабар бар боларымыз.

Ескерту: Python-да комментарий және pass операторының ара қайырмасы болатын арасында бір қайырмасы болған соң, интерпретатор комментарийді толық бас тартатады, ал pass өтілмейді.

pass операторын функция немесе клас ішінде пайдалану

def функция(аргументтер):
    pass
