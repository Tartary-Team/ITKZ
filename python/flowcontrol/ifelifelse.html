Python if…else Тілек

Компьютерлік программалау жасаушыларды if операторын пайдаланамыз, тек келесі ерекшелікке сай болдырылған уақытта блок кодты орындау үшін.

Мысалы, оқушының алған бағалары бойынша (A, B, C) бағалау.

  1. егер пайыз 90-дан асқан болса, A бағасын беру
  2. егер пайыз 75-тен асқан болса, B бағасын беру
  3. егер пайыз 65-тен асқан болса, C бағасын беру

Python-да if...else операторын үш әдепі бар.

  1. if операторы
  2. if...else операторы
  3. if...elif...else операторы

1. Python if операторы

if операторын Python-да қолданылатын синтаксисі:

if шарт: # if операторының тізбегі

if операторы шартты бақылау арқылы ойлайтын.

  1. Егер шарт True-ге айыппай болса, if операторының тізбегінің ішіндегі код орындалады.
  2. Егер шарт False-ге айыппай болса, if операторының тізбегінің ішіндегі код өткізілмейді.

Мысал 1: Python if операторы

number = 10

# тексеру, егер сан 0-дан үлкен болса
if number > 0:
    print('Сан позитивтік.')

print('Егер операторы оңай')

Шығармашылық

Сан позитивтік.
Егер операторы оңай

Атаулардың негізгі мысалында, number атандырылған өрісті құрдың. Тексеру шартын көре отырып,

number > 0

Осында, number 0-дан үлкен болып табылады, шарт True болады.

Егер number мәнінің қийметін негативті бүтін сан болса, міне, -5.

number = -5

Қазіргі кезде бағдарламаны іске өзгертсек, шығармашылық:

Егер операторы оңай

Бұл себебі number мәні 0-дан аз болғаны үшін шарт False болады. Сондықтан, if блогының төмендігі өтіледі.

2. Python if…else операторы

if операторында қосымша else бөлімі болуы мүмкін.

if...else операторының синтаксисі:

if шарт:
    # егер шарт дұрыс болса орындалатылатын код блогы

else:
    # егер шарт дұрыс емес болса орындалатылатын код блогы

if...else операторы берілген шартты бағалау:

Егер шарт **дұрыс** болса,

  • if ішіндегі код орындалады
  • else ішіндегі код өтіледі

Егер шарт **дұрыс емес** болса,

  • else ішіндегі код орындалады
  • if ішіндегі код өтіледі

Мысал 2. Python if…else операторы

number = 10

if number > 0:
    print('Позитивтік сан')
else:
    print('Негативтік сан')

print('Мысал өзгертілмейді')

Шығармашылық

Позитивтік сан
Мысал өзгертілмейді

Атаулардың жоғарыдағы мысалында, number атандырылған өрісті құрдың. Тексеру шартын көре отырып,

number > 0

Мәні number 10, ол шартты бағаланау True болады. Сондықтан, if блогының төмендігі орындалады.

Егер number мәнінің қийметін негативті бүтін сан болса, міне, -5.

number = -5

Қазіргі кезде бағдарламаны іске өзгертсек, шығармашылық:

Негативтік сан
Мысал өзгертілмейді

Осында, тексеру шарты False болады. Сондықтан, else блогының төмендігі орындалады.

3. Python if…elif…else операторы

if...else операторы екі құрамынан артық код блогын орындау үшін қолданылады.

Бірақ, егер көбінесе құрамынан артықты таңдау жасау келсе, онда if...elif...else операторын қолданамыз.

if...elif...else операторының синтаксисі:

if шарт1:
    # код блогы 1

elif шарт2:
    # код блогы 2

else: 
    # код блогы 3

Мұнда:

  1. Шарт1 true болса, код блогы 1 орындалады.
  2. Шарт1 false болса, онда шарт2 бағаланады.
    1. Шарт2 true болса, код блогы 2 орындалады.
    2. Шарт2 false болса, код блогы 3 орындалады.

Мысал 3: Python if…elif…else операторы

number = 0

if number > 0:
    print("Позитивтік сан")

elif number == 0:
    print('Нөл')

else:
    print('Негативтік сан')

print('Мысал өзгертілмейді')

Шығармашылық

Нөл
Мысал өзгертілмейді

Жоғарыдағы мысалда, number атандырылған өрісті 0-мен құрдың. Мұнда, бізге екі шартты байланыс жасауды көрсеткен:

Осында, екі жауапты шарт False болады. Сондықтан, else блогының төмендігі орындалады.

Python Таяу if операторлары

Мұнда if операторынан кейін if операторын қолдануға болады. Бұл nested if операторы деп аталады.

Nested if операторының синтаксисі:

# сыртқы if операторы
if шарт1:
    # оператор(лар)

    # ішкі if операторы
    if шарт2: 
        # оператор(лар)

Ескертпе:
- Біз ішкі if операторына else және elif операторларын қоса аламыз (көрсетілгенше).
- Егер болмаса, мысалы, сыртқы else және elif операторлары бар болса, ішкі if операторын сыртқы else және elif операторларында орналастыруымыз.
- Біз көптеген деңгейлі if операторларын таяулауымыз.

Мысал 4: Python Nested if операторы

number = 5

# сыртқы if операторы
if (number >= 0):
    # ішкі if операторы
    if number == 0:
      print('Сан 0')

    # ішкі else операторы
    else:
        print('Сан позитивтік')

# сыртқы else операторы
else:
    print('Сан негативтік')

# Шығармашылық: Сан позитивтік

Жоғарыдағы мысалда, берілген санның позитивтік, негативтік немесе 0 болатындысын тексеру үшін nested if операторын қолданамыз.
