Python Негізгі Кіру және Шығу

Python Шығу

Python-да өшірілетінді шығару үшін біз өзімізді print() функциясын пайдалана аламыз. Мысалы,

print('Python қуатты')
# Шығару: Python қуатты

Мұнда print() функциясы жолдың ішінде жаттығумен жаталған стрингді көрсетеді.

print() функциясының синтаксисті

Жоғарыдағы кодта print() функциясы тек бір параметрді алуды көрсетіп отыр.

Бірақ print функциясының жалпы синтаксисті 5 параметрді қабылдайды

print(object= separator= end= file= flush=)

Мында,

  • object - Шығарылатын мән(деректер)
  • sep (міндетті емес) - print() ішінде бірнеше object деректерді бөлу үшін пайдаланамыз.
  • end (міндетті емес) - жаттық бағдарламаларды қосуға мүмкіндік береді, мәселен: жаңа жол "\n", табуляция "\t"
  • file (міндетті емес) - мәндер көрсетілетін жер. Оның әдепкі мәні sys.stdout (экран)
  • flush (міндетті емес) - шығатын немесе буферленетіндігін көрсететін булев түрінде. Әдепкі мәні: False

Мысал 1: Python Шығару Мәліметі

print('Қайырлы таң!')
print('Бүгін жаулап')

Шығару

Қайырлы таң!
Бүгін жаулап

Жоғарыдағы мысалда, print() операторы тек object шығаратындығын көрсетеді. Мұнда end үшін мән пайдаланылмайды. Сондықтан ол әдепкі мәні '\n' түсініледі.

Сондықтан, біз шығарманы екі айырмашы жолда аламыз.

Мысал 2: Python print() келесі бағамен

# алдауымен жаттығумен
print('Қайырлы таң!', end= ' ')

print('Бүгін жаулап')

Шығару

Қайырлы таң! Бүгін жаулап

Екінші print() операторынан кейін end= ' ' мәнін қосауды көрейік.

Сондықтан, біз шығарманы бос орынмен бір жолда аламыз.

Мысал 3: Python print() сеп түрімен

print('Жаңа Жыл', 2023, 'Таныстымыз', sep= '. ')

Шығару

Жаңа Жыл. 2023. Таныстымыз

Жоғарыдағы мысалда, print() операторына бірнеше items немесе деректер көрсетіледі.

Енгізілгенде мыналарды print() операторы ішінде орнатылған сипаттамалы параметр sep= ". " пайдаланылғанын көредік.

Сондықтан, шығару мыналарды . орнатып, коммадан алдыранып көрсетеді.

Мысал: Python айнымалылар мен Литералдарды Шығару

Біз Python айнымалыларын және Python литералдарын шығару үшін print() функциясында пайдалана аламыз. Мысалы,

number = -10.6

name = "Programiz"

# литералдарды шығару     
print(5)

# айнымалыларды шығару
print(number)
print(name)

Шығару

5
-10.6
Programiz

Мысал: Стрингтерді Біріктіру

Біз print() операторы ішінде екі стрингті үйлесімді біріктіруге де боламыз. Мысалы,

print('Programiz жақсы.')

Шығару

Programiz жақсы.

Мұнда,

  • + операторы 'Programiz жақсы.' және 'awesome.' стрингтерін үйлесіп отырады
  • print() функциясы біріктірілген стрингті шығаратындығын көрсетеді

Шығару өңдеу

Кейде, шығармамызды көрінісін көмекпен өзгерту қажет болады. Бұл str.format() әдісін пайдалана отырып істейді. Мысалы,

x = 5
y = 10

print('x мәні {} және y мәні {}'.format(x,y))

Мұнда, фигуралы асаулар {} орындалған. Оларды көрсету үшін біз оларды нөмірлеу арқылы көрсете аламыз (tuple index).

Python Кіру

Программалау жасауда, бізге пайдаланушыдан кіру кізім кетуі келеді. Python-да біз input() функциясын қолдана аламыз.

input() функциясының синтаксисті

input(prompt)

Мұнда, prompt экранда көрсетілетін стринг. Оны қосу үшін жаңа пайдалану бойынша таңбалауланады.

Мысал: Python Пайдаланушы Кірімі

# пайдаланушыдан кіру кіру үшін input() пайдалану
num = input('Сан енгізіңіз: ')

print('Сіз енгіздіңіз:', num)

print('num дата түрі:', type(num))

Шығару

Сан енгізіңіз: 10
Сіз енгіздіңіз: 10
num дата түрі: <class 'str'>

Жоғарыдағы мысалда, біз input() функциясын қолдана алып, кіруге мүмкіндік бердік жасап, оны num айнымалысында сақтадык.

Енгізілген мән 10 сан емес, себебі type(num) <class 'str'> көрсетеді.

Пайдаланушыдан кіруге енгізілген мәні санга айналдыру үшін мы int() не float() функцияларын пайдалана аламыз:

num = int(input('Сан енгізіңіз: '))

Мұнда, пайдаланушыдан кіруге енгізілген деректерді санга айналдыруды үйлесімділікті пайдаланамыз.
