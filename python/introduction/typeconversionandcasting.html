Python Түрді Ауыстыру

Программалау жасаушылықта түрді ауыстыру - бұл бір түрден басқа түрге деректерді ауыстыру процесідір. Мысалы: int деректеріні str-ге ауыстыру.

Python программалау тілінде екі түрді түр ауыстыру бар.

  • Күйге кеткен Ауыстыру - автоматты түр ауыстыру
  • Мысалды Ауыстыру - қолмен жасалған түр ауыстыру

Python Күйге кеткен Түр Ауыстыру

Кейбір ситуацияларда Python автоматты түрді басқа түрге ауыстырады. Бұл күйге кеткен түр ауыстыру деп аталады.

Мысал 1: Бүтін санды бүтіндеуге ауыстыру

Python кішірек деректер түрін (бүтін сан) көп өлшемді түрге (ондай сан) ауыстырған жағдайды көрейік.

integer_number = 123
float_number = 1.23

new_number = integer_number + float_number

# жаңа мән мен алушарлық деректер түрін көрсету
print("Мән:", new_number)
print("Деректер Түрі:", type(new_number))

Шығару:

Мән: 124.23
Деректер Түрі: <class 'float'>

Жоғарыдағы мысалда, біз екі атаулы integer_number және float_number өрістерін int және float түрлерінде соответственно жасадык.

Содан кейін бұл екі атаулыны қосып, нәтижесін new_number өрісіне сақтадык.

Мынау new_number мәні 124.23 және деректер түрі float .

Себебі Python өзі деректер қайыруын жасау үшін кішірек деректер түрлерін үлкен деректер түрлеріне өзгертуді көрсетеді.

Ескерту:
- Қосылмаларды көру үшін TypeError аламыз, мысалы, '12' + 23 str және int -ді қосу көрсетіп жатамасақ болады. Python кейіпкерігі болмайды дегенде Implicit Conversion қолданбайды.
- Python-да бұл түрдегі жағдайларға шешім бар, сондай-ақ Explicit Conversion деп аталады.

Эксплицитті Түрді Ауыстыру

Эксплицитті түрді ауыстыруда пайдаланушы объекттінің деректер түрін қажетті деректер түріне ауыстырады.

Мынау, int(), float(), str() т.б., түрлеріндегі орнатылған функцияларды қолданамыз, эксплицитті түрді ауыстыруды орындау үшін.

Бұл түрді ауыстыруды өз атынан типтасу деп атайды, себебі пайдаланушы объекттінің деректер түрін (типін) ауыстырады.

Мысал 2: Стринг мен бүтін санның қосылуы

num_string = '12'
num_integer = 23

print("Type Casting бұрын num_string мәнінің деректер түрі:",type(num_string))

# эксплицитті түрді ауыстыру
num_string = int(num_string)

print("Type Casting кейін num_string мәнінің деректер түрі:",type(num_string))

num_sum = num_integer + num_string

print("Жалпы: ",num_sum)
print("num_sum мәнінің деректер түрі:",type(num_sum))

Шығару:

Type Casting бұрын num_string мәнінің деректер түрі: <class 'str'>
Type Casting кейін num_string мәнінің деректер түрі: <class 'int'>
Жалпы: 35
num_sum мәнінің деректер түрі: <class 'int'>

Жоғарыдағы мысалда, біз екі атаулы num_string және num_integer өрістерін соответственно str және int түрлерінде жасадык. Кодты қайда көретіндігімізді көздегеніміз,

num_string = int(num_string)

Мынау, num_string -ді бүтін сан түріне эксплицитті түрде ауыстыру үшін int() пайдаландық.

num_string -ді бүтін сан мәніне өзгерткен кейін Python екі атаулыны қоса алады.

Сонымен, бізге num_sum мәні 35 және деректер түрі int көрінгендігін аламыз.

Ескерту Көлемдері

  1. Түрді Ауыстыру - объекттің бір деректер түрінен басқа деректер түріне ауыстыру.
  2. Күйге кеткен Түрді Ауыстыру Python интерпретаторы автоматты түрде орындайды.
  3. Python күйге кеткен түр ауыстыруда деректерді асырайтындықпен күйіп кетеді.
  4. Эксплицитті Түр Ауыстыру өз атынан Типтасудан деп аталады, пайдаланушы орнатылған функцияларды қолдана отырып объекттің деректер түрлерін өзгертеді.
  5. Типтасудан дегенде деректерді асырайтындық болуы мүмкін, себебі біз объектті белгілі деректер түріне түрлендіреміз.
