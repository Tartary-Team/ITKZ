Python Деректер Түрлері

Компьютердік программалау арқылы деректер түрлері өрісте сақталатын деректердін түрін көрсетеді. Мысалы,

num = 24

Мұнда, 24 (бүтін санды) num атаулы переменге берілген. Сондықтан, num переменгінің деректер түрі int класына тиесілі.

Python Деректер Түрлері

Деректер Түрлері Класстар Сипаттама
Сан int, float, complex сан бойынша мәндерді сақтайды
Строка str белгілердің последовательлігін сақтайды
Последовательность list, tuple, range элементтердің тізімін сақтайды
Мэппинг dict деректерді кілт-мәні формасында сақтайды
Boolean bool тек True не False мәндерін сақтайды
Мөлшер set, frozenset уникалды элементтердің тізімін сақтайды

Себебі Python программалау жасауында барлығы объект болатын сияқты, деректер түрлері шешімдер мен өрістердің (объект) класстары.

Python Сан Деректер Түрі

Python программалау жасауында санның деректер түрі сан бойынша мәнді сақтау үшін пайдаланылады.

Бүтін сандар, жатты сан және комплекс сандар Python сандары категориясында орын алады. Оларды Python-да int, float және complex класстары деп атайды.

  • int - шектелмеген көлемді бүтін санды сақтайды.
  • float - қандай болғанда бағанда болатын децимал нүктелерді сақтайды және ол 15 децимал басқарушылығы бар.
  • complex - комплекс санды сақтайды.

Бір ауыстырма ретінде бір өріс немесе мәні күйге кеткен классын білу үшін type() функциясын пайдалана аламыз.

Мысалдарды көрейік,

num1 = 5
print(num1, 'тыпы', type(num1))

num2 = 2.0
print(num2, 'тыпы', type(num2))

num3 = 1+2j
print(num3, 'тыпы', type(num3))

Шығару:

5 - бүтін түрі <class 'int'>
2.0 - оралған түрі <class 'float'>
(1+2j) - комплекс түрі <class 'complex'>

Қалай болса, біз 5, 5.0 және 1+2j мәндері бар 3 атаулы өрістерді құрдық.

Сондықтан,

  • 5 - бүтін мән болып, type() num1 переменгінің класын int ретінде қайтарады, мынасы <class 'int'>
  • 2.0 - оралған мән болып, type() num2 переменгінің класын float ретінде қайтарады, мынасы <class 'float'>
  • 1 + 2j - комплекс сан болып, type() num3 переменгің класын complex ретінде қайтарады, мынасы <class 'complex'>

Python Тізім Деректер Түрі

Тізім бұл бірігіп тұрған бірігіпті немесе бір-біріне тең сипаттамаларды қамтынан тұруды көрсетеді, араларында коммаларды пайдаланып жатты өрістердін ішіне алады және көмегінде [ ] белгісімен жабылған. Мысалы,

languages = ["Swift", "Java", "Python"]

Мұнда, біз 3 жатты қойылған стринг мәндері бар languages аталған тізімді құрдық.

Тізім Өрістеріне Кіруге

Тізімден өрістерге кіруге, біз индекс нөмірін (0, 1, 2, …) пайдаланамыз. Мысалы,

languages = ["Swift", "Java", "Python"]

# индекс 0-дағы элементке кіру
print(languages[0])   # Swift

# индекс 2-деғі элементке кіру
print(languages[2])   # Python

Жоғарыдағы мысалда біз тізімдің ішіндегі элементтерге кіруге индекс мәндерін пайдаланбыз.

  • languages[0] - languages тізімінен бірінші элементке кіру, мәні Swift
  • languages[2] - languages тізімінен үшінші элементке кіру, мәні Python

Тізімдер туралы көбірек білу үшін Python Тізімі бетін көріңіз.

Python Tuple Деректер Түрі

Tuple - бұл тізімге орнысты, бірақ тек айырмашылығы тізімдерді анықтаудан тиесіліді. Tuple құрылғаннан кейін өзгертуге болмайды.

Python-да, біз элементтерді түпті сақтау үшін кіріс тақырыптарын пайдаланамыз () . Мысалы,

product = ('Xbox', 499.99)

Мұнда, product бір тізім болып, стринг мәні Xbox және бүтін мәні 499.99 .

Тізім Өрістеріне Кіруге

Тізімдерге ұқсас, Python программалау тілінде тізім элементтеріне кіруге индекс нөмірін пайдаланамыз. Мысалы,

# тізімді құру
product = ('Microsoft', 'Xbox', 499.99)

# индекс 0-дағы элементке кіру
print(product[0])   # Microsoft

# индекс 1-деғі элементке кіру
print(product[1])   # Xbox

Тізімдер туралы көбірек білу үшін Python Тізімдері бетін көріңіз.

Python Строка Деректер Түрі

Строка - бұл бір немесе екі бапты белгілі боларынан тұратын белгілердін тұратын последовательлік. Мысалы,

name = 'Python'
print(name)  

message = 'Python for beginners'
print(message)

Шығару:

Python
Python for beginners

Жоғарыдағы мысалда, біз стринг-түріндегі атаулы name және message перемендерін соответственно 'Python' және 'Python for beginners' мәндерімен құрдық.

Строкалар туралы көбірек білу үшін Python Строкалары бетін көріңіз.

Python Мөлшер Деректер Түрі

Мөлшер - бұл басқа басқа түсті элементтерді орнатқан біртіректі тізім. Мөлшерді орнату үшін фигураларды { } ішіне пайдаланылған мәндермен көмекке келеді. Мысалы,

# student_id атаулы мөлшерді құру
student_id = {112, 114, 116, 118, 115}

# student_id элементтерін көрсету
print(student_id)

# student_id түрін көрсету
print(type(student_id))

Шығару:

{112, 114, 115, 116, 118}
<class 'set'>

Мұнда, біз 5 бүтін мәні бар student_info аталған мөлшерді құрдық.

Мөлшерлер тізбектер емес, оларды индекске келтірудің мәні жоқ. Сондықтан, кесу операторы [] жұмыс істемейді.

Мөлшерлер туралы көбірек білу үшін Python Мөлшерлері бетін көріңіз.

Python Сөздік Деректер Түрі

Python сөздігі элементтерді орнатқан біртіректі тізім. Ол элементтерді кілт/мән жиынында сақтайды.

Мысалы,

# capital_city атаулы сөздікті құру
capital_city = {'Nepal': 'Kathmandu', 'Italy': 'Rome', 'England': 'London'}

print(capital_city)

Шығару:

{'Nepal': 'Kathmandu', 'Italy': 'Rome', 'England': 'London'}

Жоғарыдағы мысалда, біз capital_city атаулы сөздікті құрдық. Мұнда:

  1. Кілттер - 'Nepal', 'Italy', 'England'
  2. Мәндер - 'Kathmandu', 'Rome', 'London'

Кілттерді Қолданып Сөздік Мәндеріне Кіруге

Мынауларды value-ларды келесіде қолданамыз. Бірақ басқа жолмен жоқ. Мысалы,

# capital_city атаулы сөздікті құру
capital_city = {'Nepal': 'Kathmandu', 'Italy': 'Rome', 'England': 'London'}

print(capital_city['Nepal'])  # Kathmandu мәнін көрсетеді

print(capital_city['Kathmandu'])  # қате хабарламасын шығаратындау

Мұнда, біз capital_city атаулы сөздіктен кілттерді қолданып мәндерді кірегімізді көрдік.

Әрі, 'Nepal' - бұл кілт, сондықтан capital_city['Nepal'] оның мәніні Kathmandu қайтарып отырады.

Бірақ, 'Kathmandu' - бұл 'Nepal' кілтіне арналған мән, сондықтан capital_city['Kathmandu'] қате хабарламасын шығаратындау.

Сөздіктер туралы көбірек білу үшін Python Сөздігі бетін көріңіз.
