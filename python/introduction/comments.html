Python Түсініктері (Comments)

Компьютерлік программалау жасаушылардың кодты түсініктеп қалуы үшін пайдаланатын назар аудармалар.

Түсініктеме интерпретатор толықтай өткізіп тұрады. Олар теңшелген программалаушылар үшін арналған. Мысалы,

# екі айнымалыны анықтау және инициализациялау
num1 = 6
num2 = 9

# шығару результатын
print('Мұнда шығару')

Мұнда біз келесі түсініктерді пайдаланамыз,

  • екі айнымалыны анықтау және инициализациялау
  • шығару результатын

Python-да Түсініктемелердің Түрлері

Python тілінде екі түрлі түсініктемелер бар:

  • жалпы жол түсініктемесі
  • көп жол түсініктемесі

Python-да бір қатарды түсініктеме

Бір қатарды түсініктеме бір жолда басталып, оны тауып өтеді. Бір қатарды түсініктеме жазу үшін # символын пайдаланамыз. Мысалы,

# өрісті жасау
аты = 'Ерік Картман'

# мәнін басу
print(аты)

Шығармашылық

Ерік Картман

Мұнда біз екі бір қатарды түсініктеме жасадык:

  1. # өрісті жасау
  2. # мәнін басу

Бір қатарды түсініктемені кодпен бірге қолдану қатты болады.

аты = 'Ерік Картман' # аты - тізбек

Мұнда, # алдынгі код орындалады және # кейінгі код интерпретатормен жұмыс істемейді.

Python-да көп қатарды түсініктеме

Python көп қатарды түсініктеме жазу үшін аралық әдісті ұсынбайды. Бірақ, бұл мәселені шешу үшін басқа жолдар бар.

Бір қатарды түсініктеме жазу үшін көмекші # символын бірнеше қатардың басына қойа аламыз. Мысалы,

# Бұл ұзын түсініктеме
# және ол кеңейеді
# бірнеше қатарға

Мұнда, көкіректі көрсетілген көмекші қатар бірнеше қатарды түсініктеме ретінде қабылданады және оларды интерпретатор жұмыс істемейді.

Басқа бір жол - бұл ''' не """ көмекшілерін пайдалану.

Бұл көмекшілер өзге өлшемді қатарлар үшін пайдаланылады. Бірақ, оны өзге айнымалы немесе функцияға берілмесе, оларды түсініктеме ретінде пайдалана аламыз.

Интерпретатор оны өзге айнымалы немесе функцияға берілмеген жолдарды ұмытады.

Мысалды көрейік,

''' Бұл те
салқын түсініктеме
бірнеше қатарға '''

Мұнда, көмекші ұзақтыру жолымен өзге айнымалыға берілмегені үшін интерпретатор оны ұмытады. Техникалық ретте бұл көп қатарды түсініктеме емес, бірақ оны онымдарды түсініктеме ретінде пайдалана аламыз.

Python Түсініктеме Пайдалану

1. Кодты Түсініктеме Жасау

Егер біз кодымызда түсініктеме жазсақ, келесі референция үшін бұл тиімді болады.

Сонымен қатар, басқа дамытударга кодты түсінуі өте қарапайым болады.

2. Түсініктеме Пайдалану Бойынша Дебаг Жасау

Программа жүргізілген кезде қате алса, біз қате шығатын кодты алауыздырау көмекшісі ретінде түсініктеме жасай аламыз, онда оны алауыздыраудан тыс қылмайтындарымыз. Мысалы,

print('Python')

# print('Error Line )

print('Django')

Мұнда, print('Error Line) қатесін жасау кезінде қате шығатын себеппені, біз оны түсініктеме ретінде өзгерту аламыз. Қазір программа қатесіз жүргізіледі.

Түсініктемелерді дебаг құралы ретінде қалыпты пайдалану көрсеткіші.

Ескерту: Әуескертпеуші кодты айтармасқа жатпай болмау үшін көмекшілерді не боп тауып өткізу ретінде қолданыңыздар. Түсініктемелерді несіз несіге өздігіні неге жасадымызды айтармасқа пайдалану керек.
