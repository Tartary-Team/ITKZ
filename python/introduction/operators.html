Python Операторлар

Операторлар - бұл айнымалылар мен мәндерде өңдеулерді орындайтын арнайы белгілер. Мысалы,

print(5 + 6)   # 11

Мұнда, + - бұл екі саны 5 және 6 қосу жасау өңдегі оператор.

1. Python Арифметикалық Операторлар

Арифметикалық операторлар математикалық өңдеуді орындау үшін пайдаланылады - қосу, алау, көбейту, жасау жаттығулары, салыстыру, бөлу жаттығулары, көбейту жаттығулары, тамыздандыру жаттығулары жаттығулары көпшілігі. Мысалы,

sub = 10 - 5 # 5

Мұнда, - - бұл екі мән немесе айнымалуларды камтып отыратын арифметикалық оператор.

Оператор Операция Мысалы
+ Қосу 5 + 2 = 7
- Алау 4 - 2 = 2
* Көбейту 2 * 3 = 6
/ Бөлу 4 / 2 = 2
// Тамыздандыру 10 // 3 = 3
% Модуль 5 % 2 = 1
** Көтеру 4 ** 2 = 16

Мысал 1: Python-да Арифметикалық Операторлар

a = 7
b = 2

# қосу
print ('Қосымша: ', a + b)  

# алау
print ('Алау: ', a - b)   

# көбейту
print ('Көбейту: ', a * b)  

# бөлу
print ('Бөлу: ', a / b) 

# тамыздандыру
print ('Тамыздандыру: ', a // b)

# модуль
print ('Модуль: ', a % b)  

# a бойынша b көтерімі
print ('Көтеру: ', a ** b)

Шығару

Қосымша: 9
Алау: 5
Көбейту: 14
Бөлу: 3.5
Тамыздандыру: 3
Модуль: 1
Көтеру: 49

Жоғарыда көрсетілген мысалда біз бірнеше арифметикалық операторларды қолданғанмыз,

  • + - a мен b ды қосу үшін
  • - - a мен b ды алау үшін
  • * - a мен b ды көбейту үшін
  • / - a мен b ды бөлу үшін
  • // - a мен b ды тамыздандыру үшін
  • % - қалдықты алу үшін
  • ** - a бойынша b көтергісін алу үшін

2. Python Тағайындау Операторлары

Тағайындау операторлары мәндерді атауларға қосу үшін қолданылады. Мысалы,

# x-ге 5 тағайындау
var x = 5

Мұнда, = - бұл x -ке 5 мәнін тағайындау үшін тағайындау операторы.

Python-да барлық тағайындау операторларының тізімі.

Оператор Атауы Мысалы
= Тағайындау Операторы a = 7
+= Қосу Тағайындау a += 1 # a = a + 1
-= Алау Тағайындау a -= 3 # a = a - 3
*= Көбейту Тағайындау a *= 4 # a = a * 4
/= Бөлу Тағайындау a /= 3 # a = a / 3
%= Қалдықты алу Тағайындау a %= 10 # a = a % 10
**= Қасатталу Тағайындау a **= 10 # a = a ** 10

Мысал 2: Тағайындау Операторлары

# a-ге 10 тағайындау
a = 10

# b-ге 5 тағайындау
b = 5 

# a-ге a және b-дың қосымшасын тағайындау
a += b      # a = a + b

print(a)

# Шығару: 15

Мұнда, біз a мен b дың қосымшасын a -ге тағайындау үшін += операторын қолданбыз.

Оларды қажет болатын басқа тағайындау операторларында қолдана аламыз.

3. Python Салыстыру Операторлары

Салыстыру операторлары екі мән/переменді салыстырады және True немесе False мәнін қайтарып табады. Мысалы,

a = 5
b =2

print (a > b)    # Жауап: True

Мұнда, > салыстыру операторын қолдану арқылы a мен b ды салыстыру үшін пайдаланылды.

Оператор Мағына Мысалы
== Тең 3 == 5 бізге Жалған қайтаратын
!= Тең емес 3 != 5 бізге True қайтаратын
> Көп 3 > 5 бізге Жалған қайтаратын
< Аз 3 < 5 бізге True қайтаратын
>= Көп немесе тең 3 >= 5 бізге Жалған қайтаратын
<= Аз немесе тең 3 <= 5 бізге True қайтаратын

Мысал 3: Салыстыру Операторлары

a = 5

b = 2

# теңдік операторы
print('a == b =', a == b)

# теңдік емес операторы
print('a != b =', a != b)

# көптік операторы
print('a > b =', a > b)

# аздык операторы
print('a < b =', a < b)

# көп немесе тең операторы
print('a >= b =', a >= b)

# аз немесе тең операторы
print('a <= b =', a <= b)

Шығармасы

a == b = Жалған
a != b = True
a > b = Жалған
a < b = Жалған
a >= b = Жалған
a <= b = Жалған

Ескерту: Салыстыру операторлары қабылдау іске асыру және циклдерде пайдаланылады. Оларды біз соңғы нұсқаларда өтетін боламыз.

4. Python Логикалық Операторлар

Логикалық операторлар айнымалын True немесе False ма?дама бойынша тексеру үшін пайдаланылады. Олар шешу іске асыруда пайдаланылады. Мысалы,

a = 5
b = 6

print((a > 2) and (b >= 6))    # Жауап: Жалған

Мұнда, and — бұл логикалық оператор ЖӘНЕ. Себебі a > 2 мен b >= 6 көрсеткілері көптеген болды, онда нәтиже Жалған .

Оператор Мысал Мағына
and a and b Логикалық ЖӘНЕ: тек үшін обандылар барысында True
or a or b Логикалық Немесе: обандардың кем дегенде бірі True болса, онда True
not not a Логикалық ЕМЕС: операндасы False болмаса True, және көмегімен: False.

Мысал 4: Логикалық Операторлар

# логикалық ЖӘНЕ
print(True and True)     # Жалған
print(True and False)    # Жалған емес

# логикалық Немесе
print(True or False)     # Жалған

# логикалық ЕМЕС
print(not True)          # Жалған емес

Ескерту: Бұл логикалық операторлардың жауаптар кестесі бар.

5. Python Битті Операторлар

Битті операторлар операндтарды біріктірілген биттердің стройкаларымыздан көмек көрген жасау түрінде жұмыс істейді. Олар біт бойынша жұмыс істейді, сондықтан олардың атауы.

Мысалы, 2 біріктіктің байтында 10 жатамында жататып, 7 біріктіктің байтында 111 жатамында жататып көрсетілген.

Төменгі кестеде: x = 10 (0000 1010 байтында) және y = 4 (0000 0100 байтында) деп саналсақ

Оператор Мағына Мысал
& Битті Және x & y = 0 (0000 0000)
| Битті Немесе x | y = 14 (0000 1110)
~ Битті Емес ~x = -11 (1111 0101)
^ Битті XOR x ^ y = 14 (0000 1110)
>> Битті оңға жылжу x >> 2 = 2 (0000 0010)
<< Битті солға жылжу x << 2 = 40 (0010 1000)

6. Python Әрекетшілік операторлар

Пайдаланушы өзінің есепшілікті операторларды қолдайды. Оларды ақпараттау және тілекті қолдану жасау мүмкіндігімен тауарлайды. Мысалдармен бірге төменде түсіндірілгендері көрсетілген.

Теңестік операторлар

Python тілінде is және is not бұлардык жағдайда екі мәннің тізімге түсіген немесе түсіген болғанын тексеру үшін қолданылады. Егер екі теңдік болса, олардың ойынша болуы оларды теңдейтін сияқты болмайтындығын айту қажет.

Оператор Мағына Мысал
is Операндтар теңдік болса (True қайтаратынды) x is True
is not Операндтар теңдік болмаса (True қайтаратынды) x is not True

Мысал 4: Python-тағы Теңестік операторлар

x1 = 5
y1 = 5
x2 = 'Hello'
y2 = 'Hello'
x3 = [1,2,3]
y3 = [1,2,3]

print(x1 is not y1)  # Жалпысы False шығарады

print(x2 is y2)  # Жалпысы True шығарады

print(x3 is y3)  # Жалпысы False шығарады

Мысалда көрінеді, x1 мен y1 бірдей мәнді бұрыннан бұрын болғаны үшін олар тең, сондай-а олар бірдей мәнді және теңдей болатыны үшін олар идентификацияланады. x2 мен y2 (стрингтер) үшін де олар бірдей мәнге ие, сонымен қатар идентификацияланады.

Бірақ x3 мен y3 (тізімдер) бірдей мәнге ие болады, бірақ олар идентификацияланбайды. Бұл оларды сызба меншікте өзара біріккенге қараушында ойынымен көрсетуімізден көп болады.

Мысал 5: Membership операторлары Python-да

x = [1, 2, 3, 4, 5]

print(3 in x)  # 3 саны x тізімінде табылады, сондықтан True шығарады
print(6 not in x)  # 6 саны x тізімінде табылмайды, сондықтан True шығарады

Мысалда көрінеді, x тізімінде 3 саны табылады, сондықтан in операторы True шығарып, 6 саны табылмайды, сондықтан not in операторы True шығарып тұрады.

Мысал 5: Membership операторлары Python-да

x = 'Hello world'
y = {1:'a', 2:'b'}

# 'H' тауары x стрингінде бар мағынасын тексеру
print('H' in x)  # True шығарып тастайды

# 'hello' тауары x стрингінде бар мағынасын тексеру
print('hello' not in x)  # True шығарып тастайды

# '1' кілті y диктында бар мағынасын тексеру
print(1 in y)  # True шығарып тастайды

# 'a' кілті y диктында бар мағынасын тексеру
print('a' in y)  # False шығарып тастайды

Output

True
True
True
False

Мысалда, 'H' x стрингінде бар, бірақ 'hello' x стрингінде бар емес (ескеріп тұрып, Python регистрдік емес).

Ойда 1 дикте y-да кілт болып, 'a' оның мағынасы. Сондықтан 'a' in y False шығарады.
