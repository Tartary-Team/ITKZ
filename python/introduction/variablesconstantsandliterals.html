Python Айнымалылары, Сабақтамалары мен Литералдары

Python Айнымалылары

Программалау жасау кезінде айнымалы – бұл деректерді сақтау үшін қолданылатын контейнер (сақтау аймағы). Мысалы,

number = 10

Мұнда, number 10 мәнін сақтау үшін пайдаланылатын айнымады атаймыз.

Python-да Мәндерді Айнымаларға Беру

Жоғарыдағы мысалдан көріндігі дайындалғанда, мұнда мәнді айнымаларға беру үшін атауыш операторы = пайдаланамыз.

# site_name айнымасына мән беру
site_name = 'programiz.pro'

print(site_name)

# Шығармашылық: programiz.pro

Жоғарыдағы мысалда, біз site_name айнымасына 'programiz.pro' мәнін бердік. Сонымен қатар, берілген мәнді site_name айнымасына бергенімізді шығардык.

Ескерту: Python түрді білдірген тіл, сондықтан сізге айнымалы түрін көмекші операторды анықтаманы ұсынбайды. Ол автоматты түрде біледі, programiz.pro стринг деп анықталатын жасау үшін site_name айнымасын стринг түрінде айнымалайды.

Python-да Айныманың Мәнін Өзгерту

site_name = 'io.pro'
print(site_name)

# site_name айнымасына жаңа мән беру
site_name = 'apple.com'

print(site_name)

Шығармашылық

io.pro
apple.com

Мұнда, site_name айнымасының мәнін 'io.pro' дан 'apple.com' ге өзгерткіміз.

Мысал: Бірнеше мәнді бірнеше айнымаларға беру

a, b, c = 5, 3.2, 'Сәлем'

print(a)  # 5 басылады
print(b)  # 3.2 басылады
print(c)  # Сәлем басылады

Егер біз бір уақытта бірнеше айнымаларға екінші бір мән бергіміз келсе, мыналарды пайдалана аламыз:

site1 = site2  = 'prog.com'

print(site1)  # prog.com басылады
print(site2)  # prog.com басылады

Мұнда, біз site1 мен site2 айнымаларына тең бір сізбен мән 'prog.com' бердік.

Python Айнымалыларын Аталу үшін Ережелер

  • Сабақтама мен айнымалы аталуының аты аз әріптермен ( a-z ) не көбі арқылы, немесе көшбасшылармен ( A-Z ), немесе сандармен ( 0-9 ), немесе астынсыздан (**_**) тұрыптарымен болуы тиіс. Мысалы:
snake_case
MACRO_CASE
camelCase
CapWords

  • Мағанын жасауы тиіс аталуымыз. Мысалы, қатын күншіл аталуы v дан бетер.

  • Екі сөзден тұрыпты аталу үшін оларды біріктіру үшін астынсыздан (_) пайдаланымыз. Мысалы:

my_name
current_salary
  • Python көшбасшыл. Сондықтан num мен Num - бұл ерекше айнымалылар. Мысалы,
num = 5 
Num = 55
print(num) # 5
print(Num) # 55
  • Қалпына келтірілген кілт сөздерді (if, True, class жатамасын) айнымалы аты болмасы.

Python Сабақтамалары

Сабақтама - бұл өзгерту мүмкіндігі жоқ бір түрдегі айнымалы.

Python-да сабақтамалар өзгерту, функцияларды, модульдерді (переменные, функции т.б.) орналасатын, бастапқы файлға импортталатын файлдарда болады.

Біріктіріп, себепті бізді өзара файлда сабақтамаларды қалыптау және оларды бастапқы файлда қалдырамыз.

`constant.py` файлын құрыңыз:

# сабақтамаларды анықтау 
PI = 3.14
GRAVITY = 9.8

`main.py` файлын құрыңыз:

# біз құрылған constant файлын импорттау
import constant

print(constant.PI) # 3.14 басылады
print(constant.GRAVITY) # 9.8 басылады

Жоғарыда көрсетілген мысалда, біз constant.py модуль файлын құрдымыз. Сондай-ақ, біз PI және GRAVITY сабақтамаларына сабақтама мәнін бердік.

Содан кейін, біз main.py файлын құрып, constant модулін импорттадык. Сонымен қатар, біз сабақтама мәнін шығардык.

Ескерту: Жасырындай, Python-да біз сабақтамаларды пайдаланбаймыз. Оларды барлық көшбасшылардан айырап анықтау бір дайындыққа ұшырады, бірақ, ол мәнін өзара жасаушылықты тоқтаушы қылмайды.

Python Литералдар

Литералдар - бұл программада берілген мәндерді көрсетулер. Олар сандар, тақырыптар немесе тізбек мәндер болуы мүмкін. Мысалы, 'Сәлем, Әлем!', 12, 23.0, 'С', т.б.

Литералдар көбірек айнымалылар немесе сабақтамаларға мән беру үшін пайдаланылады. Мысалы,

site_name = 'prog.com'

Жоғарыда көрсетілген выраженияда site_name - бұл айнымалы, 'prog.com' - бұл литерал.

Python Санды Литералдар

Санды литералдар өзгермейтін (тұратын) екендігін айтар. Санды литералдар 3 роды нөмірлік түрді бола алатын 3 түрлі: Integer, Float, және Complex.

Түр Мысал Ескертпе
Ондік 5, 10, -68 Қалыпты сандар.
Бинарлық 0b101, 0b11 0b басталатын.
Үштік 0o13 0o басталатын.
Он шестнадцатеричный 0x13 0x басталатын.
Көлденеңді-нүктедегі Литерал 10.5, 3.14 Қалыпты нүктедегілі сан.
Комплекс Литерал 6 + 9j a + bj формасында сан, мында a - есепті және b - көмекті бөлімін алмайды.

Python Логикалық Литералдар

Екі логикалық литерал бар: True және False.

Мысалы,

result1 = True

Мұнда, True логикалық литералын result1 айнымасына бердік.

Python-да Тізбегі және Таңбалар Литералдары

Таңбалар литералдары – бұл аударманы бар қатарларда орналасқан Юникод таңбалары. Мысалы,

some_character = 'С'

Мұнда, С таңбалар литералы some_character айнымасына берілген.

Ойша, Тізбе литералдары – таңбалардың қайтарайын көмекшілігінде топталған тізбелер.

Мысалы,

some_string = 'Python жақсы'

Мұнда, ‘Python жақсы’ – бұл тізбелер литералы some_string айнымасына берілген.

Python-да Арнайы Литерал

Python-да None деген бір арнайы литерал бар. Біз оны бос айнымалыды анықтау үшін пайдаланамыз. Мысалы,

value = None

print(value)

# Шығармашылық: None

Мұнда, None шығармашылықтарының value айнымасына берілген мәні жоқ екені себепті, бізге None шығады.

Литерал Коллекциялар

Төрт түрлі литерал коллекциялары бар: Тізбе литералдар, Көрсеткі литералдар, Сөздік литералдар және Жиысты литералдар.

# тізбе литералы
fruits = ["алма", "манго", "апельсин"] 
print(fruits)

# көрсеткі литералы
numbers = (1, 2, 3) 
print(numbers)

# сөздік литералы
alphabets = {'a':'алма', 'b':'қап', 'c':'мысық'} 
print(alphabets)

# жиысты литералы
vowels = {'a', 'e', 'i' , 'o', 'u'} 
print(vowels)

Шығармашылық

['алма', 'манго', 'апельсин']
(1, 2, 3)
{'a': 'алма', 'b': 'қап', 'c': 'мысық'}
{'a', 'e', 'i', 'o', 'u'}

Жоғарыда көрсетілген мысалда, біз алмалардың тізбесін, нөмірлердің көрсеткісін, беттермен сөздіктердің алфавитін жасаған бір жиыстын жасағанмыз және үгіттердің жиыстын жасағанмыз.

Литерал коллекциялар туралы көбірек ақпарат алу үшін Python Деректер Түрлері бетіне көз жетіңіз.
