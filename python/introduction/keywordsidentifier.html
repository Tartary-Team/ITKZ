Python Кілт сөздері мен Кімімділер

Python Кілт сөздері

Кілт сөздері - бұл Python программалау тілінде алгоритмге арналған алдын ала белгілі, алдау тілінде пайдаланылатын сөздер.

Біз кілт сөзді перемендік аты, функцияның аты не басқа кімімділер түрінде пайдалана алмаймыз. Олар Python тілінің синтаксисі мен үлгісін анықтау үшін пайдаланылады.

True, False мен None кілт сөздерінден тасымалдау тартылған болатындар жаттығушы жазылуы тиіс. Барлық кілт сөздердің тізімі төменде берілген.

False await else import pass
None braek except in raise
True class finally is return
and continue for lambda try

Барлық кілт сөздерін бір арда көре отырып, оларды түсіну көптегендей бола алатын болары.

Егер толығырақ көрсетуді іздесеңіз, мысалдармен бірге барлық кілт сөздерінің толығы тізімін осында көре аласыз.

Python Кімімділер Кімімділер - перемендіктерге, сыныптарға, әдістерге т.б. берілген аттар. Мысалы,

language = 'Python'

Мұнда, language перемендік (кімімді) болып, мәні Python болады.

Біз Python-дың ішіне тұрған жасалған аттар болмас байланысты кімімділерді перемендік атауы ретінде пайдалана алмаймыз. Мысалы,

continue = 'Python'

Жоғарыдағы код қате, себебі мына жерде continue атты перемендікке пайдаланылғанын көрсеткен. Перемендіктер туралы көбірек мәлімет алу үшін Python перемендіктері бетіне қарауыңыз.

Кімімділерге атау беру үшін ережелер

  • Кімімділер кілт сөз бола алмайды.
  • Кімімділер әріптердің көптігі мен сандардың диапазоны болуы мүмкін. Бірақ оларды бірінші етіп, немесе _ теңгерімен бастау көрсету керек. Кімімдінің бірінші етіп келгені санды болмауы керек.
  • Бір кімімдігін аптап, _ теңгерімен бастау жаттығушы.
  • Ортақ бос орындар болмайды.
  • !, @, #, $ және т.б. келесі белгілерді пайдалану алмайды.

Қалпына келтіру керек Нешелер

Python - көп болмауы сипаттамалы тіл. Бұл мағыналайтын, Variable мен variable бірдей емес.

Іздеудігізге аса тұрау жасау үшін кімімділерге түсіндірім беретін ат беріңіз. c = 10 тауары, мысалы, count = 10 деп жазу, бірақ ол кодты көбейту үшін не іске қосу керек, солай деп білу көбінесе жаттығушы болады.

Бірнеше сөзді астына қоспауша сызықты пайдалану арқылы сөзділік жасау мүмкін, мысалы, this_is_a_long_variable.
